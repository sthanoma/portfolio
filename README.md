# Sajal Shrestha — Developer Portfolio Website

A modern and fully responsive personal portfolio website built using HTML, CSS, and JavaScript.  
This project showcases personal information, skills, projects, experience, education, and contact details with an interactive and animated UI design.

---

## 🌐 Live Features

- Modern dark-themed portfolio UI
- Animated gradient text effects
- Interactive custom cursor
- Smooth scrolling navigation
- Scroll reveal animations
- Responsive design for mobile and desktop
- Project showcase cards
- Timeline-based experience section
- Stylish skills section
- Contact and social links section

---
